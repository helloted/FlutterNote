# 圆角设置
> 1

单独设置圆角

```dart
          decoration: BoxDecoration(
              borderRadius: BorderRadius.only(
                  bottomLeft: Radius.circular(12.0),
                  bottomRight: Radius.circular(12.0))),
```

四个圆角一起设置

```dart
          decoration: BoxDecoration(
              borderRadius: BorderRadius.circular(20.0)),
```
